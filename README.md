# 🚀 AI Conveyor Color Segregation System

A modern full-stack web application that simulates an intelligent conveyor belt system for color-based object segregation. Built with FastAPI, MongoDB, and a dynamic JavaScript frontend.

---

## 🌐 Live Demo
https://dhanyasrig-ai-conveyor-system.hf.space

---

## 📌 Project Overview

This project simulates an industrial automation system where colored objects move across a conveyor belt, and the user must correctly classify them in real-time.

It demonstrates:
- Real-time UI interaction  
- Backend API integration  
- Database persistence  
- User authentication  

---

## 🛠️ Tech Stack

### 💻 Frontend
- HTML5  
- CSS3 (Glassmorphism UI, animations)  
- JavaScript (Game logic + API integration)  

### ⚙️ Backend
- FastAPI (Python)  
- REST API  
- JWT Authentication  

### 🗄️ Database
- MongoDB Atlas (Cloud Database)  
- Stores users and score history  

### 🚀 Deployment
- Hugging Face Spaces (Docker)

---

## 🎮 Features

- 🔐 User Login Authentication  
- 🎯 Real-time color selection gameplay  
- 📊 Score tracking & history  
- ⚡ Interactive animated UI  
- 🌐 Fully deployed web app  

---

## 🧠 How It Works

1. User logs into the system  
2. Colored items move on the conveyor belt  
3. User selects the correct color  
4. Score updates based on accuracy  
5. Data is stored in MongoDB
